# fonky-monkey
A simple Python script to send Fonkey Monkey Friday via Discord webhook using [discord.py](https://discordpy.readthedocs.io/en/stable/)

## Usage
- Clone this repo.
- Run FonkeyMonkey.py.
- Enter a valid webhook address.
